Bienvenido alnura.es

Alnura.es ofrece servicios en la web para la gestión de recursos comunitarios tanto físicos como temporales. Facilita la formación de comunidades de usuarios a las que se asocian recursos de uso común como salas de reuniones, pistas deportivas, salas multiusos o puestos de atención. Crea una web para cada comunidad creada y permite gestionar el uso de sus recursos mediante un sistema de reservas previas.
